Windows Insight 2022-2

Desktop Insight no longer uses an installer, just download this zip archive, extract it as a folder, open the folder and double-click the INSIGHT64.EXE file (the manual is also in that folder). If Windows Smart-Screen presents a dialog complaining about it being from an unidentified developer, click the More Info link and click Run Anyway. Insight requires the Mirosoft Universal Runtime DLLs, these are built into Windows 10 and kept updated. If you are using an older Windows (e.g. 7, 8) and Insight complains that a dll file is incorrect download and install the Visual Studio runtime at https://www.microsoft.com/en-us/download/details.aspx?id=48145 To uninstall Insight just trash the folder and the one Insight creates for its data (documents/insight).

Insight for DOS

The original Digitalfire Insight from the 1980s. Unpack these files, put them in a folder and mount that as the A: drive in DOSBox (or other emulator). Then switch to the A: drive and type "insight" and press Enter. Normally, F1 calls the help but that is not working so just read the Help40.txt file. Insight has a similar user interface to Lotus 1-2-3, press the first letter of items on the menu bar along the bottom, press Escape to go back.
